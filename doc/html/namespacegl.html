◆ prefix

enum gl::prefix : uint32_t
strong

Prefix of logging output.

Use bitwise or for multiple prefixes at the same time:

gl::set_prefixes(gl::prefix::FILE | gl::prefix::LINE);

Bitwise and as well as xor are also supported.

See also
set_prefix()
Enumerator
NONE 

No prefix.

FILE 

File name. For example 'main.cpp'.

LINE 

Line number in file. For example 'Line: 16'.

FUNCTION 

Function name. For example 'calculate()'.

TIME 

Current local time as hour:minute:second.millisecond. For example '10:02:13.057'.

THREAD 

ID of current thread. For example 'TID: 12'.
