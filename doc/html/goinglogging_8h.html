◆ GOINGLOGGING_NEWLINE

#define GOINGLOGGING_NEWLINE

Newline character to use after each logging message.

May e.g. be changed to std::endl to enable flush after each logging message:

#define GOINGLOGGING_NEWLINE std::endl
#include "goinglogging/goinglogging.h"
Note
Defaults to \n, i.e. no flushing.

◆ l

#define l (   ...)

Log variables to stdout.

Use as:

int i = 1;
l(i);
std::string s("s");
l(i, s);

Which outputs:

i = 1, s = s
Note
Supports up to 16 variables on the same line.
Uses prefix information set with set_prefixes().
See also
l_arr()
l_mat()
set_prefixes().

Log array to stdout.

Parameters
vArray to print.
nNumber of elements.

Used as:

int a[] = {3, 4, 5};
l_arr(a, 3);

Which outputs:

a: [0] = 3, [1] = 4, [2] = 5
Note
Uses prefix information set with set_prefixes().
Warning
Behaviour is undefined if n is larger than the allocated array.
